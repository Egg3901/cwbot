## Summary

Implement a complete ticketing system for the Corporate Warfare Discord bot. Users can create support tickets that spawn private channels, staff can claim and manage tickets, and all interactions are logged.

---

## Acceptance Criteria

- [x] `/ticket` slash command opens ticket creation modal
- [x] Ticket categories: Support, Bug Report, Suggestion, Other
- [x] Creates private channel visible only to user + staff role
- [x] Ticket channel includes close, claim, transcript buttons
- [x] Close ticket archives channel and logs transcript
- [x] Claim assigns staff member to ticket
- [x] Ticket logs sent to configured log channel
- [x] Configurable staff role and log channel
