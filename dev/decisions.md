# Architecture Decisions

## ADR-001: JavaScript over TypeScript
**Date:** 2024-12-30
**Status:** Accepted
**Context:** Initial project setup
**Decision:** Use JavaScript (ES6+) instead of TypeScript
**Rationale:** Faster iteration, simpler setup for Discord bot

## ADR-002: Modular Feature Architecture
**Date:** 2024-12-30
**Status:** Accepted
**Context:** Need to support multiple features and future API integration
**Decision:** Each feature (tickets, welcome, starboard) is a self-contained module
**Rationale:** Easier maintenance, clear separation of concerns, ready to scale
